# CogHood Resident Identity Workshop

This directory is the source-visible seed layer for persistent CogHood resident identity in `x86ml`. It is generated by `scripts/x86ml_resident_workshop.py` and should be regenerated through that script rather than edited casually by hand.

The workshop does not run Bochs, execute bundled binary artifacts, or train a model. It records the **contract** that future resident-specific LLM, reservoir, mailbox, or Bochs-trace integration loops can consume safely.

| Path | Purpose |
|---|---|
| `manifest.json` | Global workshop registry for all nine dove9 residents. |
| `residents/*.identity.json` | Per-resident compact identity seeds. |
| `iterations/seed-iteration.jsonl` | First deterministic identity-carving event for every resident. |

Each resident identity seed includes a preferred x86ml salience profile, a deterministic point in the 2,300-point Bochs configuration manifold, memory-channel emphasis, control-surface affinity, and a groundhog countermeasure. The artifacts are small on purpose: they are the overlooked stones that make recovery possible after context loss.

To regenerate the workshop artifacts, run:

```bash
python ./scripts/x86ml_resident_workshop.py
```

To validate the contract, run:

```bash
python -m unittest discover -s tests -v
```
