# CogHood Resident Identity Workshop

This directory is the source-visible seed and accumulation layer for persistent CogHood resident identity in `x86ml`. The seed contract is generated by `scripts/x86ml_resident_workshop.py`; bounded memories are integrated by `scripts/x86ml_identity_accumulator.py`. Both steps should be regenerated through those scripts rather than edited casually by hand.

The workshop does not run Bochs, execute bundled binary artifacts, or train a model. It records the **contract** that future resident-specific LLM, reservoir, mailbox, OpenCog, virtual-hardware, or Bochs-trace integration loops can consume safely.

| Path | Purpose |
|---|---|
| `manifest.json` | Global workshop registry for all nine dove9 residents. |
| `residents/*.identity.json` | Per-resident compact identity seeds. |
| `iterations/seed-iteration.jsonl` | First deterministic identity-carving event for every resident. |
| `memory/seed-memories.jsonl` | Reviewed bounded memory impressions for all nine residents. |
| `iterations/identity-accumulation.jsonl` | Derived digest-chained memory integration log. |
| `state/*.identity-state.json` | Portable accumulated identity state for each resident. |

Each resident identity seed includes a preferred x86ml salience profile, a deterministic point in the 2,300-point Bochs configuration manifold, memory-channel emphasis, control-surface affinity, and a groundhog countermeasure. Each resident identity state then adds event counts, channel counts, salience totals, the last memory digest, and an accumulated identity hash. The artifacts are small on purpose: they are the overlooked stones that make recovery possible after context loss.

To regenerate the workshop seed artifacts and then accumulate reviewed memories, run:

```bash
python ./scripts/x86ml_resident_workshop.py
python ./scripts/x86ml_identity_accumulator.py
```

To validate the contract, run:

```bash
python -m unittest discover -s tests -v
```
